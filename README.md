# Diffuse Fujii-Oren-Nayar - Godot
This is an addon for Godot 4.4+ that adds the `Fujii-Oren-Nayar-Approximation` node to the visual shader system. This node outputs the `Diffuse Light` based on [Yasuhiro Fujii approximation](https://mimosa-pudica.net/improved-oren-nayar.html) to the `Full-Orenâ€“Nayar Reflectance Model`.
